# 常见报错排查

## 1. 错误：ModuleNotFoundError: No module named 'yaml'

### 解决方法：
1. 确保已安装 pyyaml，运行 pip install pyyaml 进行安装。

## 2. 错误：ModuleNotFoundError: No module named 'dotenv'

### 解决方法：
1. 确保已安装 python-dotenv，运行 pip install python-dotenv 进行安装。

## 3. 错误：文件路径无法找到或无权限访问

### 解决方法：
1. 确保程序执行的路径正确，并且当前用户有读写权限。
